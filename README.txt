== Description ==

The Guest-post plugin will provide functionality of adding custom post type posts from frontend side with essential validations for author user role only in draft mode. Only admin can publish it after reviewing it.

It also shows the list of pending status posts through shortcode and gutenberg block.

The shortcode for add post form to show : [guest-posts]

The shortcode for display pending posts to show : [pending-guest-posts]
